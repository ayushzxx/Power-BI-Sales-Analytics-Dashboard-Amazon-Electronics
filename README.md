
# 📊 Amazon Electronics Sales & Customer Engagement Dashboard | Power BI

An interactive **Power BI dashboard** built using an Amazon Electronics Sales & Customer Engagement dataset. This project focuses on sales performance, customer engagement, pricing strategy, discounts, and product performance through interactive visualizations and KPIs.

---

## 📌 Project Overview

The objective of this project is to analyze Amazon electronics sales data and derive actionable business insights using Power BI.

The dashboard enables users to:
- Monitor key business KPIs
- Analyze sales performance across product categories
- Compare Best Seller vs Non-Best Seller products
- Evaluate the impact of discounts on revenue
- Compare Sponsored vs Organic products
- Analyze Coupon effectiveness
- Filter the dashboard dynamically using slicers
